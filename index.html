[.com/nellore/rail]
is a spliced aligner by researchers at JHU that scales to analyze many RNA-seq samples at once on Amazon Elastic MapReduce. It also works on IPython Parallel clusters and single computers.

Install the latest version (0.2.1: changelog) for the current user in one command by entering the following at a Bash prompt on an OS X or Linux box:

Copied.
(INSTALLER=/var/tmp/$(cat /dev/urandom | env LC_CTYPE=C tr -cd 'a-f0-9' | head -c 32); curl -L http://tinyurl.com/rail-0-2-1 -o $INSTALLER; python2 $INSTALLER -m || true; rm -f $INSTALLER)

For more flexibility, download the installer and follow these instructions.

Now read the preprint, review the tutorial, see how Rail makes it easy to analyze dbGaP-protected data in the cloud, or learn about other features.
